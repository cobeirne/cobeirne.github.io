Twitter Typeahead

Bus Hop | Auto suggest PTV bus routes

  • Chris O'Beirne
  • Twitter Typeahead

The Bus Hop system requires users to select their desired bus route, however there are hundreds of local bus routes within Victoria Australia. Displaying this list and finding the desired route in a traditional drop-down control is inefficient and clutters up the user interface. The system needed a search feature like Google Search.

In my role as programmer for the Bus Hop project, I solved this problem in two parts. Firstly, by querying the Public Transport Victoria (PTV) API for a dataset of bus routes within Victoria. And secondly by using the Twitter Typeahead control to auto suggest bus routes as the user enters their route name via a free text input.

This solution significantly increased the chances of users finding their desired route within a few seconds. Further improvements can be made by including bus route numbers in the Twitter Typeahead suggestions.
