Bootstrap

Bus Hop | Develop UX Framework.

  • Chris O'Beirne
  • Bootstrap

The Bus Hop UX was derived from use cases provided by our client. There were several key themes:

  • Where do I want to go?
  • When do I want to leave?
  • What is my journey?

In my role as programmer for the Bus Hop project, I nominated Bootstrap 3 as a suitable platform for the Bus Hop web application. This was because of its responsive design which would maximise compatibility across different fixed and mobile devices. In the first stages of development I mocked up several sample web pages to help the team visualise the proposed layout and navigation. It also helped us identify the need for some advanced functions such as auto suggestion to maintain the simple and clean user experience required.

The result is a demonstration of how use cases can positively influence interface design, rather than allowing the design to evolve from functional requirements alone. I plan to invest more time in use cases in future user interface design work.
