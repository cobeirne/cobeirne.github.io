Bus Hop Project

About the Project

Bus Hop is a web application that provides bus stop departure information to public transport users. It also accepts GPS location updates via a RESTFUL API for real time tracking functions. It has been developed as a Deakin University Capstone Project for which I was the principal programmer amongst a team of four members. A Bus Hop prototype has now been completed and is ready for beta testing.

As principal programmer I was responsible for the development of the Bus Hop user interface and server-side functionality. My key contributions are detailed in the following blog posts:

  • Post 03/08/2017: Develop UX Framework
  • Post 04/08/2017: Find the user’s closest bus stop.
  • Post 10/08/2017: Auto suggest PTV bus routes.
  • Post 11/08/2017: Reverse geolocate user address.
  • Post 26/08/2017: Render bus routes and map markers.
  • Post 01/09/2017: Set user location using Auto Complete.
  • Post 09/09/2017: Display dynamic user and bus map markers.
  • Post 23/09/2017: Estimate actual bus departure times.

Some of these contributions have been built on established skills such as Bootstrap, MVC and C#. However, it has been necessary to develop new skills in the areas of JavaScript, jQuery, geolocation, Google Maps API and RESTFUL web services. These new skills have been acquired through independent online study and have been essential to the successful completion of the Bus Hop project thus far. Consequently, my involvement in this project has inspired me to develop more integrated web applications in the future.
